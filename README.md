
# 🛍️ Shopix – Modern E-Commerce Website

A modern and responsive e-commerce website built using **HTML, CSS, JavaScript, and Bootstrap 5**.
This project focuses on creating a clean shopping experience with interactive UI elements, product filtering, shopping cart functionality, dark/light mode, responsive layouts, and smooth animations.

🌐 **Live Demo:**
[Shopix Live Website](https://sanarahmani1404.github.io/E-commerce-project/?utm_source=chatgpt.com)

---

## ✨ Features

* 🛒 Add to Cart functionality
* 💳 Checkout / Payment page
* 🌙 Dark & Light mode toggle
* 📱 Fully responsive design
* 🎞️ Smooth animations & transitions
* 🔍 Product categories & filtering
* ❤️ Wishlist / favorite style UI
* 🖼️ Modern product cards & carousel
* ⚡ Fast and lightweight frontend
* 🎨 Clean and professional UI/UX

## 🚀 Getting Started

To run the project locally:

1. Clone the repository

```bash
git clone https://github.com/sanarahmani1404/E-commerce-project.git
```

2. Open the project folder

```bash
cd E-commerce-project
```

3. Run the project

Simply open `index.html` in your browser.
